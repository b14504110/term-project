一、程式目的與特色
傳統物理課程中的拋體運動通常假設物體僅受重力作用，忽略空氣阻力與環境因素，然而在實際情況下，空氣阻力、風速以及物體形狀都會對運動軌跡產生顯著影響。本程式透過數值方法引入這些現實因素，使模擬結果更貼近真實物理行為。
本程式的主要特色如下：
支援 無空氣阻力（解析解） 與 含空氣阻力（數值解） 的拋體運動模擬
可選擇 線性阻力 或 平方阻力模型
引入 風場（水平與垂直風速），以相對速度計算阻力
透過 彈體形狀設定（Sphere、Cylinder、Bullet、Custom改變阻力係數與迎風面積
提供 互動式操作面板，可即時調整參數並觀察軌跡變化
支援 Euler 與 Runge–Kutta 四階（RK4） 兩種數值積分方法以比較精度差異
具備動畫播放與結果比較功能，提升模擬的直觀性
二、程式架構
程式整體可分為三個主要模組概念：
1.物理模型與數值計算（NumPy）
使用 NumPy 儲存與運算狀態向量 
建立重力、空氣阻力與風場的加速度模型
以 Euler 法與 RK4 法數值解聯立微分方程：
​2.視覺化與互動介面（Matplotlib）
主視窗顯示拋體的二維運動軌跡
右側面板提供滑桿與選單以調整物理與數值參數
X 軸下方設置控制按鍵（Reset、Play、Update、Compare）
支援動畫播放，動態呈現拋體運動過程
可同時顯示解析解與數值解以進行比較
3.控制邏輯與事件處理
Slider 變更：修改參數但不立即播放動畫
Update：重新計算並顯示完整軌跡
Play：以動畫方式呈現運動過程
Reset：回復預設設定並清空模擬狀態
Compare：顯示或隱藏無阻力解析解軌跡
此設計使「參數設定」、「數值計算」與「結果呈現」彼此分離，提升程式的穩定性與可操作性
將抽象的微分方程轉化為具體的數值與圖形結果
本模擬器不僅適合作為物理與工程教育的輔助工具，也可作為數值分析與科學計算的入門示例。
